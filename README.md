# transformando_texto_em_audio
Este projeto foi desenvolvido como parte do módulo **"Os Pilares Formais da IA: do Machine Learning à IA Generativa"** do bootcamp **Bradesco - GenAI &amp; Dados** na plataforma DIO. O objetivo é demonstrar a aplicação prática de bibliotecas de síntese de voz (Text-to-Speech) de forma simples e eficiente.
